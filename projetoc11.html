Objetivo do Projeto:

Na aula 11, você aprendeu como adicionar animações/imagens e dimensioná-las no jogo.

Neste projeto, você colocará em prática o que aprendeu na aula para criar um plano de fundo de mar com navio. O navio terá animação de movimento para cima e para baixo.

História:

Johanna está planejando fazer uma viagem. Mas devido à Covid-19, ela não vai poder ir. Agora ela está planejando criar um jogo em que possa viajar de navio e apreciar paisagens diferentes.

Você pode ajudar Johanna a projetar o jogo? A seguir está o resultado disso em ação:

*Isso é apenas para sua referência. Esperamos que use sua própria criatividade no projeto.

Primeiros Passos:

  2. Abra o editor VS Code:
  3. Clique em Arquivo e selecione Abrir Pasta…:
  4. Selecione a pasta do Modelo de Projeto (Project Template):
  5. Comece a escrever o código no arquivo sketch.js:
Tarefas específicas para completar o projeto:
  1. Na função preload():
    • Carregue a animação para mover o navio para cima e para baixo.
    • Carregue a imagem do mar.
  2. Crie um sprite para o mar e adicione uma imagem a ele de forma que o plano de fundo se pareça com o mar aberto, conforme mostrado no resultado.
    • Atribua movimento ao plano de fundo, dando velocidade X.
  3. Crie um sprite para o navio e adicione a animação do navio a ele.
  4. Clique uma vez em para verificar se está funcionando.
